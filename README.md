# 技术栈

## 后端

- [Laravel](https://laravel.com/) + [Reverb](https://laravel.com/docs/12.x/reverb)（WebSocket）

## 前端

- React + [Next.js](https://nextjs.org)
- [shadcn](https://ui.shadcn.com/)（UI） + tailwind
- Zustand（状态管理） + Immer
- Zod（字段验证）
- react-hook-form（表单填写）
- Canvas / Three.js
- (Lucide)[https://lucide.dev/]（图标）
